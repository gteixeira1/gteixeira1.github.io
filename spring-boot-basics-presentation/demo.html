Spring Framework

  • Created by Rod Johnson in 2003
  • Free and Open source. Maintained by Pivotal
  • Inversion of control container (IoC)
  • Different modules for different proposal

Why Spring Framework?

  • Enhanced developer productivity
  • Free & Open source
  • Modules
  • Who want to go back to EJB?

Why Spring Boot?

  • Setup and configuration is simpler than Spring
  • Less use of XML
  • Microservices

What is Spring Boot?

  • Learn Spring is much easier with Spring Boot
  • Bootstrap version of a spring application
  • Convention over Configuration
